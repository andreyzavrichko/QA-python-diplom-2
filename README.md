# QA-python-diplom-2

Автотесты для REST API сервиса объявлений **QA Desk** (учебный стенд Практикума).  
Стек: Python · pytest · requests · Allure · Faker.

---

## Структура проекта

```
QA-python-diplom-2/
├── api/
│   ├── __init__.py
│   ├── ad_api.py          # Методы для работы с объявлениями
│   └── user_api.py        # Методы регистрации и авторизации
├── assertions/
│   ├── __init__.py
│   └── assertions.py      # Кастомные ассерты (структура ответа, JWT)
├── tests/
│   ├── files/
│   │   └── garage.jpg     # Тестовое изображение для создания объявления
│   ├── __init__.py
│   ├── test_auth.py        # Тесты авторизации
│   ├── test_create_ad.py   # Тесты создания объявления
│   ├── test_delete_ad.py   # Тесты удаления объявления
│   ├── test_registration.py# Тесты регистрации
│   └── test_update_ad.py   # Тесты редактирования объявления
├── conftest.py             # Фикстуры pytest
├── data.py                 # Константы: URL, статус-коды, тела запросов/ответов
├── helpers.py              # Генерация тестовых данных (Faker)
├── pytest.ini              # Конфигурация pytest + Allure
├── requirements.txt        # Зависимости
└── .gitignore
```

## Покрытие тестами

| Модуль             | Тест                                             | Ожидаемый статус |
|--------------------|--------------------------------------------------|-----------------|
| Регистрация        | Успешная регистрация нового пользователя         | 201             |
| Регистрация        | Повторная регистрация существующего пользователя | 400             |
| Авторизация        | Успешная авторизация                             | 201             |
| Объявление: Create | Создание объявления со случайной категорией      | 201             |
| Объявление: Update | Успешное редактирование своего объявления        | 200             |
| Объявление: Update | Редактирование чужого объявления                 | 401             |
| Объявление: Delete | Успешное удаление своего объявления              | 200             |

## Требования

- Python 3.12+
- pip

---

## Установка

```bash
git clone <repo-url>
cd QA-python-diplom-2
pip install -r requirements.txt
```

---

## Запуск тестов

```bash
# Все тесты
pytest

# Конкретный модуль
pytest tests/test_create_ad.py

# С подробным выводом
pytest -v
```

---

## Allure-отчёт

```bash
# Запустить тесты и собрать результаты
pytest

# Сгенерировать и открыть отчёт
allure serve allure-results
```

Результаты сохраняются в директорию `allure-results/` (задано в `pytest.ini`).

## Конфигурация

Все URL и тестовые данные вынесены в `data.py`:

| Переменная               | Значение                                              |
|--------------------------|-------------------------------------------------------|
| `BASE_URL`               | `https://qa-desk.stand.praktikum-services.ru/api/`    |
| `EXISTING_USER_EMAIL`    | `test12@test.test`                                    |
| `EXISTING_USER_PASSWORD` | `123456`                                              |
| `CATEGORY_LIST`          | `["Авто", "Садоводство", "Хобби", "Книги", "Технологии"]` |

## Фикстуры

| Фикстура                              | Назначение                                            |
|---------------------------------------|-------------------------------------------------------|
| `get_access_token`                    | Авторизует пользователя, возвращает Bearer-заголовок  |
| `random_category_in_request_create_ad`| Возвращает тело запроса с рандомной категорией        |
